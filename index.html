PenBank

PenBank is a modern, responsive landing page for a banking website built using React.js and Tailwind CSS. This project showcases my skills in front-end development, including component-based architecture, state management, and responsive design principles. The landing page features a sleek and intuitive user interface, ensuring an optimal user experience across various devices. Key functionalities include an attractive hero section, service highlights, customer testimonials, and a call-to-action section encouraging user engagement.

Chat Application

Developed a real-time chat application leveraging Firebase for backend services, React.js for the frontend, and Express with Node.js for the server-side logic. This project highlights my proficiency in building full-stack applications with a focus on real-time data synchronization and RESTful API integration. Key features include user authentication, chat rooms, message storage, and real-time updates, providing a seamless and interactive communication platform for users. The application emphasizes scalability, security, and user-friendly design.
